# FTL Naming And Layout

## Naming

- Use kebab-case.
- Prefix by subsystem or component.
- Be specific enough to avoid collisions.

## Layout

- Keep related strings together.
- Use comments for groups when helpful.
- Do not use tabs for indentation.

## Examples

- `shine-device-popup-no-power`
- `comp-shine-widget-examine-charged`
