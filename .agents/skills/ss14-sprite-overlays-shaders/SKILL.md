# SS14 Sprite Overlays Shaders

This is the rendering-resource companion to `ss14-graphics-generic-visualizer-appearance`.

## Rules

- Put SS14-ART-CORE textures under `Resources/Textures/_Art`.
- Preserve asset attribution.
